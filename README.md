# Supply Chain Correlation Analysis

This project analyzes supplier performance data to identify relationships between key supply chain metrics.

## Analysis Overview
We analyzed 69 procurement transactions to understand interactions between:
- Supplier Lead Time
- Inventory Levels
- Order Frequency
- Delivery Performance
- Cost Per Unit

## Files
- `correlation_analysis.py`: Script to perform analysis and generate visualizations.
- `correlation.csv`: Matrix of correlation coefficients.
- `heatmap.png`: Visualization of the correlation matrix.
